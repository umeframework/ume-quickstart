【运行方法(I) - 使用Eclipse启动】

  1 依赖环境
    Eclipse版本4.5或以上; JDK版本1.7或以上; Apache Maven版本3.3.3或以上; Eclipse Jetty 3.9.0插件 (Eclipse Marketplace搜索Jetty后安装)

  2.选择项目后右键点击，弹出菜单中选择 [Run With Jetty]运行。

【服务测试】
  打开浏览器窗口(浏览器推荐使用Firefox或Chrome，安装Json-handle插件以编译查看JSON格式的数据)，
  输入 http://localhost:8080/ume-quickstart-webapp 打开默认页面(查看当前Web服务的状态)。
